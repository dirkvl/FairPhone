## Random unsorted information

### Back connector

When you are looking from the back of the phone you see 5 pogo pins. With a multimeter and some trial&error you can find the following info:

```
(1)  (2)
  (3)
(4) (5)

1) vbus
2) no idea
3) gnd
4) d+
5) d-
```
